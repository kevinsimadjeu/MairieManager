# MairieManager
A web application for municipal management, designed to administer various city services and improve the interaction between the town hall and citizens. ⚠ For confidentiality reasons, the backend is not included in this repository. Only the front-end is available.

# PROJECT OVERVIEW
This application was designed to simplify the management of municipal services. It provides a simple, intuitive, and responsive interface that allows administrative staff to perform multiple tasks through the integrated modules.

# MAIN FEATURES (Front-End)
Modern and responsive interface

Dashboard for easy navigation

Management of records (birth, death, marriage, etc.) on the UI side

Ergonomic forms for data entry

Smooth navigation and reusable components

---> Important: Data-related actions (creation, modification, authentication, storage, etc.) were implemented on the server side but are not included in this GitHub repository to comply with confidentiality requirements. The front-end has been adapted to function without these private endpoints.

# TECHNOLOGIES USED (Front-End)
HTML5 / CSS3 / JavaScript /Php

Responsive Design

Custom UI components

# BACKEND (Not Included)
The backend, developed to handle:
